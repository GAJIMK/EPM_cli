# 📑 EPM (Easy employee Payment Manager)

Key1. 임직원의 복지비, 활동비 등 여러 경비를 가시적이고 편리하게 작성하고 저장할 수 있습니다. <br/>
Key2. 관리자는 임직원들의 사용 이력을 전체적으로 조회하고 승인할 수 있습니다. <br/>
Key3. 상급자와 하급자의 승인 절차를 손쉽게 해결할 수 있습니다. <br/>
